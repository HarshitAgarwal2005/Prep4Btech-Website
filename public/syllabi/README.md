# RTU Syllabus Files

This directory contains the RTU (Rajasthan Technical University) syllabus files for Computer Science Engineering.

## File Structure

- `year1-sem1.pdf` - First Year, Semester 1 Syllabus
- `year1-sem2.pdf` - First Year, Semester 2 Syllabus
- `year2-sem3.pdf` - Second Year, Semester 3 Syllabus
- `year2-sem4.pdf` - Second Year, Semester 4 Syllabus
- `year3-sem5.pdf` - Third Year, Semester 5 Syllabus
- `year3-sem6.pdf` - Third Year, Semester 6 Syllabus
- `year4-sem7.pdf` - Fourth Year, Semester 7 Syllabus
- `year4-sem8.pdf` - Fourth Year, Semester 8 Syllabus

## Upload Instructions

1. Enable "Upload Mode" in the RTU Syllabus section
2. Click the "Upload" button next to the desired semester
3. Select the PDF file for that semester's syllabus
4. The file will be uploaded and made available for download

## File Requirements

- Format: PDF only
- Size: Maximum 10MB per file
- Naming: Files will be automatically named based on year and semester

## Developer Access

Only authorized developers can upload syllabus files. The upload functionality is protected and requires proper authentication.
